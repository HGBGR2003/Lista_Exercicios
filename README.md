## 🚀 Projetos

### 🌍 Mundo Wumpus

Um projeto baseado no clássico problema de Inteligência Artificial. Um agente inteligente navega em um grid tentando encontrar o ouro e sair vivo, desviando de poços e do Wumpus — utilizando lógica proposicional e inferência para tomar decisões.

- 🔗 **Link do site:** [mundowumpus.vercel.app](https://mundowumpus.vercel.app/)
- 🧠 **Disciplina:** Inteligência Computacional
- ⚙️ **Tecnologias:** JavaScript, HTML, CSS
